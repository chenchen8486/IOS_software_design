1.6 地图工作区（Map Workspace）

对应原始手册：## 15. Map Workspace

一、功能层级与界面总览

工作区入口说明：地图工作区（Map Workspace）是 IOS 的一个独立工作区，并非弹窗。教员需通过控制面板底部页脚（1.2.7）最右侧的 Workspace 按钮打开工作区切换菜单，选择「Map / 地图」进入本界面。进入后，当前显示器将切换为地图工作区，显示地图标题栏、主区域、右侧工具栏与地图页脚四大部分。

二、核心功能详解

2.1 地图标题栏（Map Header）

地图标题栏位于页面顶部，通过一系列选项卡分组显示与飞机相关的各类信息。教员可以快速监控训练中的关键参数。

选项卡/信息组内容说明
重量与平衡
Weight & Balance		显示当前飞机重量与重心相关参数（同 8.10 重量与平衡滑动面板）
性能数据
Performance Data		显示起飞/进近性能数据，如 Stab Trim、Flaps、V1/VR/V2/VREF 等
高度、航向、指示空速
Altitude / Heading / IAS		显示飞机当前高度、航向和指示空速
通信
Communications		显示无线电通信状态和频率信息
已激活和待处理的故障
Armed & Active Malfunctions		显示当前预位或已激活的故障列表
当前时间
Current Time		显示模拟器当前时间
课程与练习信息
Lesson & Exercise Info		显示当前课程计划和训练阶段信息

主区域是虚拟空间的 2D/3D 表示，显示飞机位置、飞行路径及环境组件（其他活动实体、导航台、机场、风暴等天气要素）。

2.2.2 径向菜单（Radial Menus）

主界面上使用多种径向菜单，可通过点击地图主界面区域或特定对象访问。

光标径向菜单（Cursor Radial Menu）

点击地图空白处▼

允许教员执行以下操作：

  • 在光标位置放置一个蜘蛛网（Spider Web）
  • 将飞机重新定位到地图上的特定位置
  • 在地图上的特定位置添加风暴
  • 在地图上的特定位置添加实体

操作步骤：点击地图区域上的所需位置 → 光标放置于点击位置 → 光标径向菜单按钮显示。

图10 光标径向菜单

飞机径向菜单（Aircraft Radial Menu）

点击飞机▼

允许教员在飞机周围放置一个蜘蛛网。

操作步骤：在地图主界面上点击飞机 → 点击「附加蜘蛛网（Attach Spiderweb）」按钮 → 飞机周围显示蜘蛛网。

图11 飞机径向菜单

导航台径向菜单（Navaid Radial Menu）

点击导航台图标▼

允许教员执行以下操作：

  • 将飞机重新定位到导航台上方，同时保持其高度和航向，并显示故障导航台
  • 对于 VOR-DME 等多组件导航台，可以仅使一个系统失效
  • 重置故障导航台：当故障激活时，「故障导航台」按钮变为「重置导航台」；地图上的导航台图标指示其正常或故障状态
图12 导航台径向菜单

机场径向菜单（Airport Radial Menu）

点击机场图标▼

允许教员在机场周围放置一个蜘蛛网。

风暴径向菜单（Storm Radial Menu）

点击风暴图标▼

允许教员执行以下操作：

  • 激活一个风暴
  • 删除一个活跃的风暴
  • 编辑/更改风暴参数或属性
  • 在钩子位置添加一个风暴

2.4 地图页脚（Map Footer）

地图页脚位于页面底部，分为左、中、右三段控件。

2.4.1 左侧页脚（Left Side Footer）

左侧页脚提供关于学员表现的补充信息（如图表），包括事件、绘图、清除三个切换/卷起菜单。

图19 左侧页脚
事件（Events）

「事件」切换菜单显示/隐藏系统识别的超限和异常情况信息。

图20 左侧事件切换菜单
绘图（Plots）

「绘图」卷起菜单显示以下五类图表，用于在地图上叠加显示并评估机组表现。教员选择对应类型后，系统会在主区域地图上渲染其关联图表。

图表类型评估目的包含的具体指标 / 图表
精密进近
Precision Approach		 通过监控飞机在三个精密进近图表上的实际位置，评估机组进近表现 • Glide Path Deviation（下滑道偏差）
• Localizer Deviation（航向道偏差）
• Speed Deviation（速度偏差）
非精密进近
Non-Precision Approach		 通过监控飞机在三个非精密进近图表上的实际位置，评估机组进近表现 • Height Above Terrain (HAT) vs Distance to Touchdown（HAT 与距接地点距离）
• Center Line Deviation（中心线偏差）
• Speed Deviation（速度偏差）
起飞
Takeoff		 通过监控飞机在三个起飞图表上的实际位置，评估机组起飞表现 • Height Above Terrain (HAT) vs Distance to Takeoff（HAT 与距起飞点距离）
• Center Line Deviation（中心线偏差）
• Speed Deviation（速度偏差）
尾部间隙
Tail Clearance		 通过监控五个关键参数，评估尾部间隙以避免尾部擦地（Tail Strike） • Airspeed（空速）
• Radar Altitude（雷达高度）
• Tail Clearance (inches)（尾部间隙，单位：英寸）
• Pitch Attitude（俯仰姿态）
• Pitch Rate（俯仰速率）
跑道
Runway		 通过监控飞机在跑道上的实际位置，评估机组跑道运行表现 飞机在跑道上的实际位置（Runway Position）
清除绘图（Clear Plots）

点击「清除绘图」按钮可立即清除当前地图上显示的所有图表，恢复主区域地图的纯净视图。

风暴（Storm）与风暴属性滑动面板

点击中间页脚「风暴」按钮显示风暴属性滑动面板（STORM PROPERTIES Sliding Panel）。该面板允许教员在地图环境中添加多种类型的风暴、旋转所选风暴以及为所选风暴添加各种效果。面板包含以下三个主要区域：

  • 风暴列表（List of Storms）：显示模拟中已激活或未激活的风暴
  • 类型（TYPE）：显示所有可插入模拟的风暴类型
  • 旋转与效果（ROTATION & EFFECTS）：选择风暴类型后显示，提供编辑风暴所需的全部控制项

添加风暴的两种方式：

方式一：通过光标径向菜单
  1. 在地图主界面上点击目标位置，放置光标。

  3. 编辑风暴属性（旋转、效果、位置等）。
  4. 点击「激活（Activate）」按钮激活风暴。
  5. 所选风暴旁边出现红色「激活（ACTIVE）」标志，表示该风暴已在模拟中激活。
  6. 中间页脚「风暴」按钮上覆盖红色徽章，指示当前已激活的风暴数量。

流程说明：地图工作区教员核心操作流程展示了教员进入地图工作区后的单次完整操作动线。流程以「进入 Map Workspace」为起点，教员首先选择操作域，完成对应操作后直接汇聚到「操作完成」终点。

教员可从以下六大操作域中选择：

  1. 标题栏监控：查看飞机重量与平衡、性能数据、高度/航向/空速、通信状态、故障列表、当前时间以及课程与练习信息，快速掌握训练中的关键参数。所有信息均为实时监控，无需额外操作即可获取。
  2. 主区域交互：查看风向标（实时风向）和罗盘（支持 North Up / Heading Up 切换），或通过 5 种径向菜单执行具体操作：
    • 光标径向菜单：放置蜘蛛网、飞机重定位、添加风暴/实体
    • 飞机径向菜单：在飞机周围放置蜘蛛网
    • 导航台径向菜单：飞机重定位至导航台上方、故障/重置导航台
    • 机场径向菜单：在机场周围放置蜘蛛网
    • 风暴径向菜单：激活、删除、编辑风暴参数或添加至钩子位置
  3. 控制地图视图：使用右侧工具栏调整地图显示，包括缩放（+/-）、自动范围（确保飞机与参考跑道同屏）、比例尺（5 ~ 350 nm）、透视（3D 模式 + 双拨盘调整视角）、地图定位（中心定位至机场/飞机/风暴）、符号（T 图标控制文本标签 / 眼睛图标控制元素显示）以及指示器（眼睛图标控制蜘蛛网、FMS 航线、时间线、航迹、罗盘、比例尺、风羽的显示/隐藏）。
  4. 左侧页脚：查看 Events（系统识别的超限和异常信息）、Plots（5 类性能评估图表：精密进近、非精密进近、起飞、尾部间隙、跑道）或使用 Clear 清除时间线、风暴、事件列表和航迹。
  5. 中间页脚：使用 Marker 在飞行路径上标记彩色事件；通过 Storm 打开风暴属性面板执行「选择类型 → 编辑属性（旋转/效果/位置） → 激活/删除」的子流程；控制 Recording 开始/停止录制（需配备简报/讲评系统）；或使用 Print 打印当前页面。
  6. 右侧页脚：使用 Latch 将地图锁定到本机保持居中；切换 2D/3D 显示模式；打开 Help 获取帮助；Workspace 按钮可切换至其他 IOS 工作区并结束当前地图会话。

六个操作域最终均汇聚至「操作完成」终点；若教员通过右侧页脚 Workspace 切换至其他 IOS 工作区，则视为结束当前地图会话。如需执行下一项任务，教员可重新从 Map Workspace 入口启动新流程。
