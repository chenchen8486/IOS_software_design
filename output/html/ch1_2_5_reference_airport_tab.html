二、核心功能详解

2.1 主界面参数字段速览

参考机场选项卡包含 14 组参数输入文本字段，点击后显示相应的滑动面板或键盘。部分参数支持 Wheel Picker（长按或上下滑动呼出）。

字段名称显示内容交互方式弹出面板/键盘
机场（Airport）当前参考机场 ICAO 代码点击 / Wheel Picker机场与重新定位滑动面板（详见本页 2.2 节）
跑道（Runway）参考机场当前跑道号点击 / Wheel Picker机场与重新定位滑动面板（详见本页 2.2 节）
时间（Time）当前时刻点击 / Wheel Picker时刻键盘 / Wheel Picker 选择黎明/白天/黄昏/夜间（详见本页 2.3 节）
地面风（磁）（Surface Wind）参考机场和跑道当前风况点击 / 长按风向/风速字段 + Wheel Picker地面风滑动面板（详见本页 2.4 节）
能见度（Visibility）当前云下能见度点击 / Wheel Picker能见度键盘（详见本页 2.5 节）
RVR当前跑道视程点击 / Wheel Picker能见度/RVR 滑动面板（详见本页 2.5 节）
降水（Precipitation）参考机场当前降水情况点击 / Wheel Picker降水滑动面板（详见本页 2.6 节）
云层（Clouds）参考机场/飞行场当前云层状况点击 / Wheel Picker云层滑动面板（详见本页 2.7 节）
温度（Temperature）参考机场当前温度点击 / Wheel Picker参考温度键盘（详见本页 2.8 节）
ISA 偏差（ISA Dev）当前 ISA 温度偏差点击 / Wheel PickerISA 偏差键盘（详见本页 2.8 节）
QNH当前机场海平面气压点击 / Wheel PickerQNH 键盘（详见本页 2.8 节）
微暴流/风切变（Microburst/Windshear）当前激活的微暴流/风切变剖面点击风切变与微暴流滑动面板（详见本页 2.9 节）
跑道状况（Runway Condition）参考机场当前跑道状况点击 / Wheel Picker跑道状况滑动面板（详见本页 2.10 节）
交互式机场（Interactive Airport）机场跑道图形点击跑道直接更改参考跑道并更新跑道文本字段

警告：导航数据库每 28 天更新一次。更新后，所有基于旧导航数据库创建的飞行计划将不再可用，必须重新创建。

3.4 数据流与开发流程推导

3.4.1 数据流

教员操作 → UI 事件 → 状态变更 → 模拟器指令 → 反馈呈现

【参考机场变更】
教员点击「机场」字段 → 弹出机场与重新定位面板
  → 选择新 ICAO 代码和跑道 → 更新参考机场状态
    → 向模拟器发送 AIRPORT_CHANGE 指令
      → 更新地图、FMS、跑道照明、风切变位置等关联模块

【环境条件变更】
教员修改地面风/能见度/降水/云层/温度等
  → 校验边界条件（温度范围、互斥关系）
    → 更新环境状态对象
      → 向视觉系统发送 ENV_UPDATE 指令
        → 更新视景渲染、仪表显示、ATIS 消息

【重定位执行】
教员选择重定位预设 → 点击「重新定位」
  → 设置 FLIGHT_FREEZE = true
    → 向模拟器发送 REPOSITION 指令（含预设 ID、目标坐标、飞机配置参数）
      → 模拟器执行位置/姿态/速度瞬移
        → 完成后 FLIGHT_FREEZE = false

【跑道状况变更】
教员选择污染物（冰/雪/积水等）+ 粗糙度
  → 校验温度范围与降雨状态（互斥条件）
    → 更新 RUNWAY_CONDITION 状态
      → 向模拟器发送 RUNWAY_FRICTION_UPDATE 指令
        → 模拟器调整轮胎-跑道摩擦系数模型
          → 影响刹车距离计算、侧滑风险判定、ATIS 消息生成

【FMS 飞行计划保存/召回】
教员输入描述/密码 → 校验密码长度（≤10 字符）
  → 向 FMS 发送 FLIGHTPLAN_SAVE/RECALL 指令
    → FMS 校验导航数据库版本兼容性
      → 保存/加载飞行计划至 FMS 内部存储器
        → 返回 save_status / load_status
          → 更新 FMS 内存列表显示

【地面操作执行】
教员选择推出/轮挡/引导员/跑道状况
  → 校验飞机状态（是否移动/离地/已放置轮挡）
    → 发送 GROUND_OPS 指令
      → 模拟器执行相应动画与状态变更
        → 更新飞机位置/舱门状态/地面服务设备状态
