二、核心功能详解

2.1 主界面参数字段速览

参考机场选项卡包含 14 组参数输入文本字段，点击后显示相应的滑动面板或键盘。部分参数支持 Wheel Picker（长按或上下滑动呼出）。

字段名称显示内容交互方式弹出面板/键盘
机场（Airport）当前参考机场 ICAO 代码点击 / Wheel Picker机场与重新定位滑动面板（详见 2.2）
跑道（Runway）参考机场当前跑道号点击 / Wheel Picker机场与重新定位滑动面板（详见 2.2）
时间（Time）当前时刻点击 / Wheel Picker时刻键盘 / Wheel Picker 选择黎明/白天/黄昏/夜间
地面风（磁）（Surface Wind）参考机场和跑道当前风况点击 / 长按风向/风速字段 + Wheel Picker地面风滑动面板（详见 2.4）
能见度（Visibility）当前云下能见度点击 / Wheel Picker能见度键盘（详见 2.5）
RVR当前跑道视程点击 / Wheel Picker能见度/RVR 滑动面板（详见 2.5）
降水（Precipitation）参考机场当前降水情况点击 / Wheel Picker降水滑动面板（详见 2.6）
云层（Clouds）参考机场/飞行场当前云层状况点击 / Wheel Picker云层滑动面板（详见 2.7）
温度（Temperature）参考机场当前温度点击 / Wheel Picker参考温度键盘（详见 2.8）
ISA 偏差（ISA Dev）当前 ISA 温度偏差点击 / Wheel PickerISA 偏差键盘（详见 2.8）
QNH当前机场海平面气压点击 / Wheel PickerQNH 键盘（详见 2.8）
微暴流/风切变（Microburst/Windshear）当前激活的微暴流/风切变剖面点击风切变与微暴流滑动面板（详见 2.9）
跑道状况（Runway Condition）参考机场当前跑道状况点击 / Wheel Picker跑道状况滑动面板（详见 2.10）
交互式机场（Interactive Airport）机场跑道图形点击跑道直接更改参考跑道并更新跑道文本字段

2.2 机场与重新定位滑动面板（AIRPORT & REPOSITION Sliding Panel）

触发方式：点击主界面「机场」或「跑道」参数字段弹出。

图：机场与重新定位滑动面板

此滑动面板允许教员查看并选择参考/目的地机场和跑道、将飞机重新定位到预设位置、执行地面操作、调整跑道灯光强度。包含四个选项卡：参考位置、机场重新定位、地面操作、FMS 快速设置。

2.2.1 参考位置选项卡（REFERENCE LOCATION Tab）

属性说明
参考机场只读文本字段，显示参考机场 ICAO 代码和活动跑道编号
目的地机场只读文本字段，显示目的地机场 ICAO 代码和活动跑道编号
更改参考按钮点击展开参考机场（位置）滑动面板，显示可用机场 ICAO 代码列表及跑道
更改目的地按钮点击展开目的地机场（位置）滑动面板，显示可用机场 ICAO 代码列表及跑道
自动参考位置复选框切换自动/手动模式。自动模式：参考机场自动更新至最近可用跑道；手动模式：除非教员或课程计划输入，否则不更改

2.2.2 机场重新定位选项卡（AIRPORT REPOSITION Tab）

包含四个部分：等待/滑行、登机口/停机位、重定位预设、跑道照明。

图：机场重新定位选项卡（典型）

等待/滑行部分（HOLD / TAXI）

互斥单选 ▼
预设飞机配置边界行为
等待（HOLD）起落架放下；襟翼保持（若当前位置对起飞有效）否则襟翼 5；舱门关闭重新定位结束时设置飞行冻结，停用加速
滑行（TAXI）起落架放下；襟翼保持（若当前位置对起飞有效）否则襟翼 5；舱门关闭重新定位结束时设置飞行冻结，停用加速

登机口/停机位部分（GATE / PARKING SPOT）

互斥单选 ▼

点击所需单选按钮将飞机重新定位到登机口/停机位。飞机配置：起落架放下、襟翼收上、减速板收起、1L 舱门及右前/右后货舱门打开。重新定位结束时设置飞行冻结，停用加速。

2.2.5 FMS 内存保存/召回滑动面板（FMS MEMORY SAVE/RECALL Sliding Panel）

图：FMS 内存保存/召回滑动面板

此滑动面板允许教员保存当前 FMS 飞行计划或将先前保存的飞行计划加载到 FMS 内存中。包含三个选项卡：消息、密码保护、未受保护。以及 SIMSOFT CONTROLS 选项卡（特定机型专用）。

控件功能说明
描述（Description）点击输入文本字段，显示键盘输入待保存飞行计划的描述
密码（Password）点击输入文本字段，显示键盘输入最多 10 个字符的有效密码
保存（Save）点击启动保存所选 FMS 内存飞行计划的过程
召回（Recall）点击调取所选 FMS 内存飞行计划
取消（Cancel）点击取消当前输入
删除所选（Delete Selected）点击删除所选 FMS 内存飞行计划
清除所有未受保护（Clear all Unprotected）点击清除「未受保护」选项卡下列出的所有 FMS 内存飞行计划
密码保护选项卡列出所有可保存/调取的密码保护飞行计划编号，显示 FMC 飞行计划描述
未受保护选项卡列出所有可保存/调取的未受保护飞行计划编号，显示 FMC 飞行计划描述
SIMSOFT CONTROLS — 保存向 FMS 发送命令，将输入的飞行计划副本保存至 FMS 内部存储器（每次只能保存一个，不保存在模拟主机中）
SIMSOFT CONTROLS — 召回召回存储在 FMS 存储器中的飞行计划
禁止自动召回（Inhibit Auto Recall）激活时，在特定机场/跑道保存的飞行计划在重新定位至同一位置后不会自动被召回

警告：导航数据库每 28 天更新一次。更新后，所有基于旧导航数据库创建的飞行计划将不再可用，必须重新创建。

3.3 数据流与开发流程推导

3.3.1 数据流

教员操作 → UI 事件 → 状态变更 → 模拟器指令 → 反馈呈现

【参考机场变更】
教员点击「机场」字段 → 弹出机场与重新定位面板
  → 选择新 ICAO 代码和跑道 → 更新参考机场状态
    → 向模拟器发送 AIRPORT_CHANGE 指令
      → 更新地图、FMS、跑道照明、风切变位置等关联模块

【环境条件变更】
教员修改地面风/能见度/降水/云层/温度等
  → 校验边界条件（温度范围、互斥关系）
    → 更新环境状态对象
      → 向视觉系统发送 ENV_UPDATE 指令
        → 更新视景渲染、仪表显示、ATIS 消息

【重定位执行】
教员选择重定位预设 → 点击「重新定位」
  → 设置 FLIGHT_FREEZE = true
    → 向模拟器发送 REPOSITION 指令（含预设 ID、目标坐标、飞机配置参数）
      → 模拟器执行位置/姿态/速度瞬移
        → 完成后 FLIGHT_FREEZE = false

【风切变/微暴流激活】
教员选择剖面 + 强度 + 位置 → 点击「激活」
  → 更新 WINDSHEAR_PROFILE 状态
    → 向大气模型注入风切变数据（核心位置、半径、风速变化曲线）
      → 飞机飞越核心区域时动态计算局部风速矢量
