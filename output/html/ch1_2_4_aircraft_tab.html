二、核心功能详解

2.1 参数输入文本字段速览（Parameter Input Text Fields）

飞机选项卡主区域上半部分集中显示 5 个重量与平衡相关参数。所有参数均支持键盘输入和滚轮选择器两种修改方式。

参数功能定义交互方式与其他功能关系
总重 GW以所选单位（千克/磅 x1000）显示飞机总重点击弹出总重键盘，或使用滚轮选择器（详见 2.5 重量与平衡滑动面板）与 ZFW、燃油动态联动；修改 GW 时 ZFW 自动调整，若超出限制则燃油自动调整
总重重心 GW CG以 % MAC 显示总重重心点击弹出总重重心键盘，或使用滚轮选择器（详见 2.5）与 ZFW CG 动态联动；修改 GW CG 时 ZFW CG 自动调整，若超出限制则燃油减少、ZFW 增加
零燃油重量 ZFW以所选单位（千克/磅 x1000）显示零燃油重量点击弹出 ZFW 键盘，或使用滚轮选择器（详见 2.5）与 GW、燃油动态联动；修改 ZFW 时 GW 自动变化，若超出限制则燃油自动调整
ZFW CG以 % MAC 显示零燃油重量重心点击弹出 ZFW CG 键盘，或使用滚轮选择器（详见 2.5）与 GW CG 动态联动；修改 ZFW CG 时 GW CG 自动变化
燃油 FUEL以所选单位（千克/磅 x1000）显示飞机总燃油量点击弹出燃油键盘，或使用滚轮选择器（详见 2.5）与 GW、ZFW 动态联动；修改燃油时 GW 自动变化，若超出限制则 ZFW 自动调整；燃油按飞机燃油分配计划分配
滚轮选择器通用操作：长按参数直至滚轮选择器显示，或向上/向下滑动参数呼出滚轮。滚动选择后，点击绿色勾号确认，或点击红色 X 取消。

总重键盘显示当前总重量，并允许教员通过键盘或滑块输入另一个总重值。

按钮功能定义呈现方式交互方式
DL 数据链（Data Link）显示数据链下行消息状态当有来自机组的下行消息时，按钮亮起绿色点击弹出数据链滑动面板（详见 2.3）
UL 上行链路（Uplink）显示上行链路消息状态当有教员发送给机组的上行消息时，按钮亮起橙色点击弹出数据链滑动面板（详见 2.3）
VHF 通信（VHF Comm）显示飞行员/副驾驶无线电通信状态和调谐频率机组讲话时，飞行员/副驾驶图标旁显示绿色发射波点击弹出通信滑动面板（详见 2.4）
重量与重心（Weight & CG）快速访问重量与平衡设置标准按钮点击弹出重量与平衡滑动面板（详见 2.5）
服务与舱门（Services & Doors）快速访问地面服务和舱门控制标准按钮点击弹出服务与舱门滑动面板（详见 2.6）
故障（Malf）快速访问故障预选和激活按钮上红色标记表示已激活和待处理的故障数量点击弹出故障滑动面板（详见 2.7）
断路器（CB）快速访问断路器控制按钮上红色标记表示跳闸/弹出的断路器数量点击弹出断路器滑动面板（详见 2.8）
故障总结（Malf Summary）快速访问所有故障和断路器的总体状态按钮上红色标记表示故障总结中包含的故障数量点击弹出故障总结滑动面板（详见 2.9）

2.3 数据链（DL/UL）滑动面板（Data Link Sliding Panel）

数据链滑动面板允许教员向模拟飞机发送上行消息、显示模拟飞机与地面站之间的通信、查看并回复来自模拟飞机的消息，以及控制 ATC 和 ADS 连接。

图 7：数据链滑动面板（Data Link Sliding Panel）
属性说明
触发方式点击主界面 DL 或 UL 快速访问按钮弹出
面板结构包含两个一级选项卡：日志（Log）、连接控制（Connection Control）
日志选项卡（Log）显示模拟飞机与地面站之间的通信历史，最新消息始终显示在页面顶部。底部包含三个操作按钮：清除一条、清除全部、发送上行链路
连接控制选项卡（Connection Control）提供 ATC 和 ADS 中心的连接/断开控制，包含 ATC 部分、ADS 部分、地面部分

2.3.1 日志选项卡内嵌功能 — 新上行链路（New Uplink）

在日志选项卡底部点击发送上行链路按钮后，当前视图切换为新上行链路选项卡（New Uplink Tab），允许教员组装并发送上行消息。该子页面包含以下四部分：

属性说明
组（Group）列出所有上行链路消息的分组类别
场景（Scenario）列出所选组内的所有消息类别，点击场景按钮可查看该类别下的上行链路消息
场景消息（Scene Message）列出所选场景（类别）下所有可用的上行链路消息
消息编辑器（Message Editor）显示已选的上行链路消息，允许教员设置时间、高度等参数，并控制消息发送顺序
添加按钮（Add）点击上行链路消息旁的添加按钮，将该消息加入消息编辑器
移除按钮（Remove）点击消息编辑器中消息旁的移除按钮，将该消息从编辑器中移除
▲ / ▼ 按钮（Up / Down）调整消息编辑器中消息的优先级顺序，决定发送给机组的先后顺序

2.3.2 连接控制选项卡（Connection Control Tab）

连接控制选项卡提供连接/断开与 ATC 和 ADS 中心的控制功能，并显示地面通信状态。面板自上而下分为三个区域：ATC、ADS、地面。

图 9：连接控制选项卡（Connection Control Tab）
2.3.2.1 ATC 部分（ATC Section）

本节主要由只读字段和单选按钮组成。

属性说明
只读字段（RO Fields）显示当前 ATC 连接航班号、活动 ATC ID 和下一个权限 ID
自动 ATC 管理（Auto ATC Management）数据权限转移由地面站模拟自动处理，无需教员操作
自动预选（Auto Preselect）软件建议下一个权限，教员点击发送即可启动转移
手动选择（Manual Select）教员通过显示的输入字段手动选择下一个权限
断开按钮（Disconnect）点击断开当前活动的 ATC 连接，弹出确认菜单
图 10：ATC 部分（ATC Section）
2.3.2.2 ADS 部分（ADS Section）

自动相关监视（ADS）部分显示 ADS 信息并控制 ADS 连接，包括只读字段和一个 ADS 中心 ID 列表。

属性说明
只读字段（RO Fields）显示最近一次接收到的 ADS 报告：飞机 ID、纬度、经度、高度、ADS 基本报告模式
ADS 中心 ID 列表（ADS Center ID List）列出所有 ADS 中心 ID 及其相关控件
ID显示 ADS 中心 ID
报告间隔（Report Interval）显示 ADS 报告间隔
修改（Modify）点击可修改 ADS 报告间隔
状态（Status）显示 ADS 连接的状态
连接/断开（Connect / Disconnect）点击可发起或断开 ADS 连接
图 11：ADS 选项卡（ADS Tab）
2.3.2.3 地面部分（Ground Section）

此部分提供地面通信的状态，并允许教员触发通信故障。

属性说明
VHF 地面故障（VHF Ground Fault）切换按钮：正常状态点击触发与 VHF 地面站的数据链通信故障；激活状态点击清除故障并返回正常

2.4 通信滑动面板（Communication Sliding Panel）

通信滑动面板允许教员建立 VHF/HF 无线电通信、与内部通信系统（ICS）通信、收听无线电导航信号、选择各种通信模式。

图 12：通信滑动面板（Communication Sliding Panel）
属性说明
触发方式点击主界面 VHF 快速访问按钮弹出
面板结构包含五个选项卡：音量（Volume）、无线电（Radio）、内部通话（Intercom）、导航（Navigation）、特殊功能（Special Features）
音量选项卡（Volume）通过滑块控制无线电/内部通话耳机音频、导航音频信号、耳机音频、背景噪音音量，范围从最小值到最大值
无线电选项卡（Radio）显示所有可用 VHF 和 HF 电台，每个电台包含电台名称、所选频率、调谐地面站类型。每个电台有三个按钮：发射（Transmit）、接收（Receive）、SELCAL
内部通话选项卡（Intercom）允许教员与 ICS 通信，包括驾驶舱（FLT）、客舱（CAB）、旅客广播（PA）。每个 ICS 包含发射和接收按钮；若无线电发射机无电源，发射按钮禁用
导航选项卡（Navigation）允许监听 NAV、DME、Marker Beacon、ADF、ILS 信号。每个导航接收机显示接收机名称、调谐频率、接收台站名称
特殊功能选项卡（Special Features）提供私密通信、通信监听、各类呼叫及硬件控制功能（详见 2.4.1）

图 14：RADIO 选项卡（Radio Tab）

2.5 重量与平衡滑动面板（Weight & Balance Sliding Panel）

重量与平衡滑动面板允许教员快速配置飞机重量。它包含所有必要的控制功能，可快速将飞机设置为预定义重量、调整零燃油重量、调整燃油重量、管理每个燃油箱的燃油重量。

此滑动面板包含三个主要选项卡：

  • 快速设置（Quick Set）
  • 性能数据（Performance Data）

3.4 数据流与开发流程推导

3.4.1 数据流（参数修改与生效）

教员点击参数输入文本字段或快速访问按钮
    ↓
IOS 弹出对应滑动面板 / 键盘
    ↓
教员输入新值或选择新状态
    ↓
点击确认（绿色勾号）或激活按钮
    ↓
前端校验输入值范围（Dynamic Min/Max）
    ↓
校验通过 → 发送参数变更指令至模拟器核心
    ↓
模拟器核心判断功能类型
    ├─ 重量与平衡参数 → 重量与平衡计算引擎
    │   ├─ 未超出限制 → 更新所有关联参数并同步至模拟器
    │   └─ 超出限制   → 自动调整燃油或ZFW以保持合规
    ├─ 故障操作       → 故障管理系统
    │   ├─ 立即激活   → 实时注入模拟器
    │   └─ 预位       → 写入触发条件表，等待判据满足后自动激活
    └─ 通信/数据链    → 音频路由系统 / 数据链通信系统
        → VHF/HF/ICS 信号或上行链路消息发送至驾驶舱
    ↓
模拟器反馈当前参数值至 IOS 显示
    ↓
IOS 更新对应文本字段、状态标志、快速访问按钮指示灯
